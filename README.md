# readr
A website to help users track their reading list, progress, and history

---

# Features
1. Add a book to the list
- Title
- Author
- Genre
- Pages
- Read?
- Owned?
- ...more

2. Link places to buy book or check out if not owned
- Amazon
- AbeBooks

3. Reading Log
- Rating
- Place for a writeup
- Date started/date finished/pages per day

4. Visualizations
- Pages read per day
- Most common genres
- 1 Year ago today...
